# TD Bank Enterprise Risk Analytics Project

This project analyzes TD Bank’s failed First Horizon acquisition using enterprise risk analytics techniques including:

- Statistical risk modeling
- Machine learning classification
- Monte Carlo simulation
- Key Risk Indicators (KRI)
- Scenario-based risk assessment

## Key Risks
- Regulatory & Compliance Risk
- Strategic Growth Risk
- Reputational Risk

## Methods
- Lognormal, Normal, Poisson distributions
- Random Forest classification
- 10,000-iteration Monte Carlo simulation

## Tools
Python, Pandas, Scikit-learn, Matplotlib
